# Welcome to the podekex!
this project was designed using ReactJS.

## How to start.
- Clone the project.
- Run `yarn`.
- Run `yarn start`.

## How to use.
- Type a pokemon name on the form.
- Click the search button.
- Enjoy your pokemon.
